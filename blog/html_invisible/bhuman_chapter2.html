下载BHuman2015代码

你有多种方式获得BHuman代码。

git clone

选择一个合适的文件夹，在终端中直接输入以下命令：

git clone https://github.com/bhuman/BHumanCodeRelease.git

直接下载压缩包

登录BHuman的Github仓库。点击download zip下载。

注意：请不要将在windows下解压的文件夹移入linux系统。


编译前的准备工作

在终端中输入以下命令安装所需的库

sudo apt-get install qt4-dev-tools libglew-dev libxml2-dev clang graphviz

在BHumanCodeRelease文件夹中执行以下命令配置naoqi-sdk

./Install/installAlcommon /path/to/naoqi-sdk-2.1.4.13-linux32.tar.gz

将/path/to替换成自己的路径
