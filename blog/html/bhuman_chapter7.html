矫正相机

首先将BHumanCodeRelease-master/Config/Locations/Default和BHumanCodeRelease-master/Config/Locations/Default下的所有文件复制到BHumanCodeRelease-master/Config/Robots/ <your robot name>下，打开SimRobot,连接到远程机器人，打开Console ,Remote/image/lower和Remote/image/upper 以下操作建议使用网线连接进行


CameraSettings

在Console中敲get representation:UpperCameraSettings ，会的到如下回显

set representation:UpperCameraSettings camera = upper;

autoExposure = { value = 0; };

autoWhiteBalance = { value = 0; };

contrast = { value = 60; };

exposure = { value = 150; };

fadeToBlack = { value = 0; };

gain = { value = 90; };

hue = { value = 0; };

saturation = { value = 240; };

sharpness = { value = 2; };

whiteBalance = { value = 4639; };

手动更改里面的值，之后将光标移动到末尾，回车即可实现修改，主要修改exposure和gain两个值，修改到可以分辨出不同颜色即可，这两个值过大会产生较多噪点。

自动白平衡:在Console中敲dr module:CameraProvider:DoWhiteBalanceLower,回车，敲dr module:CameraProvider:DoWhiteBalanceUpper ，回车，如果对自动白平衡的效果不满意，可在上一步中手动修改。

需要保存的时候Console中敲 save representation:UpperCameraSettings，回车save representation: LowerCameraSettings，回车
